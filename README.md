# Human-Robot Interaction Laboratory

This repository contains my practical work from the Human-Robot Interaction Laboratory course.

The lab involved practical robotics exercises using ROS, Franka Emika robotic manipulator, NAO humanoid robot, teleoperation, robot kinematics, impedance control, verbal interaction, and object recognition.

## Final Course Report
The team final course report is included as `Final_Report_Team1.pdf`.

## Tasks

### Task 1: ROS Introduction and Robot Motion
Worked on ROS basics, Python scripting, publisher/subscriber nodes, Franka Emika robot startup, joint states, robot motion, and forward kinematics.

### Task 2: Kinematics and Interaction
Worked on inverse kinematics, Cartesian task-space motion, impedance/admittance control, collision detection, and force interaction.

### Task 3: Cooperative Lifting
Implemented ROS services and studied cooperative human-robot lifting using force-based interaction.

### Task 4: Teleoperation
Worked on gamepad-based teleoperation, force sensor integration, ROS parameters, launch files, Arduino integration, and haptic feedback.

### Task 5: Verbal Interaction with NAO
Developed verbal interaction using NAO, Choregraphe, QiChat, speech recognition, and non-verbal gestures.

### Task 6: Object Recognition
Worked on NAO vision recognition and YOLO-based object detection using neural networks.
